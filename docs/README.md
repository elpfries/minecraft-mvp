# docs/ — visuels du projet

Ce dossier contient les images utilisées par le README.

## Ajouter la capture d'écran (`screenshot.png`)

1. Lance le jeu : `npm run dev` puis ouvre http://localhost:5173
2. Clique dans la fenêtre pour capturer la souris, place-toi devant un joli
   paysage (colline, forêt, montagne au loin…).
3. Appuie sur **Échap** pour libérer la souris, puis fais une capture d'écran :
   - macOS : **⌘ + ⇧ + 4** (sélection) ou **⌘ + ⇧ + 3** (plein écran)
   - Windows : **Win + Maj + S**
4. Enregistre l'image ici sous le nom **`screenshot.png`**.
5. Dans le [README](../README.md), **décommente** la ligne `![Aperçu du jeu]…`.

## Optionnel : un GIF animé (`demo.gif`)

Un court GIF de 5–10 s rend le dépôt encore plus vivant. Outils gratuits :
- macOS : [Kap](https://getkap.co/)
- Windows : [ScreenToGif](https://www.screentogif.com/)

Enregistre-le sous `demo.gif` et référence-le dans le README de la même façon.

> Astuce : garde des fichiers légers (< 2–3 Mo) pour que la page se charge vite.
